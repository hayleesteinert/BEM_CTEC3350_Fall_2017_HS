Typography

Fonts

There are three different fonts loaded onto the site.
Font Information URL
Karla Font is decalared in base CSS. It is used for heading text. https://fonts.googleapis.com/css?family=Roboto|Karla:400,700
Roboto Font is decalared in base CSS. It is used for all text but headings. https://fonts.googleapis.com/css?family=Roboto
FontAwesome Font is imported from it's own style sheet that has been acquired from FontAwesome. It is used for all icons throughout the site. http://fontawesome.io

Display Headings

Traditional heading elements are designed to work best in the meat of your page content. When you need a heading to stand out consider using a display heading - a larger, slightly more opinionated heading style.

Inline Text Elements

For quoting blocks of content from another source within your document. Wrap <blockquote class="blockquote"> around any HTML as the quote. Add a <footer class="blockquote-footer"> for identifying the source. Wrap the name of the source work in <cite>.
Example

Suspendisse ut sapien consequat ipsum scelerisque iaculis vitae id velit.

Somone famous in Source Title 

<blockquote class="blockquote">
<p>Suspendisse ut sapien consequat ipsum scelerisque iaculis vitae id velit. </p>
<footer class="blockquote-footer">Somone famous in <cite title="Source Title"> Source Title</cite>
</footer>
</blockquote>
